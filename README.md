# crypy

Crypy is a cipher based web app written in TypeScript using React. 

## local setup 

```bash
    npm install
    npm run dev
``` 

## build

```bash 
    npm run build
    npm start
```

## test

```bash 
    npm run test
```

## deployment 

Currently, the project is manually deployed on Heroku. In future, a CI/CD pipeline could be set too.
